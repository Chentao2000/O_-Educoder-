## 计算机网络的定义：
- 计算机网络是 以 “相互共享资源的方式互联起来的自治计算机系统的集合”     

具有以下特征：

（1）组件计算机网络的主要目的是实现计算机资源的共享和信息交互     
 (2) 联网计算机系统是相互独立的自治系统
 (3) 联网计算机之间的通信必须遵循共同的网络协议

 ## 基本的网络拓扑：
- 基本的网络拓扑有五种，星状，环状，总线型，树状，与网状
##### 星状拓扑：
- 节点通过点与点通信线路与中心节点连接
- 中心节点控制全网的通信，任何两节点之间的通信都要通过中心节点
- 星状拓扑结构简单，易于实现，便于管理
- 网络的中心节点是全网性能与可靠性的瓶颈，中心节点的故障可能造成全网瘫痪
##### 环状拓扑：
- 节点通过点-点通信线路连接成闭合环路
