
## 学习笔记:
```
头结点的好处
- 第一: 有了头结点之后 ，插入和删除的数据元素的算法都统一了！不再需要判断是否在第一个元素之前插入或删除第一个元素
- 第二：无论链表是否为空，其头指针是指向头结点的非空指针，链表的头指针不变，因此空表和非空表的处理也就统一了！
```

```
静态链表中的指针又称为游标，指示下一个元素在数组的下标！
```
