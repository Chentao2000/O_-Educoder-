## 学习笔记:
```
停止-等待协议的信道最大利用率：

    (L/C)
 ----------
  (L/C + 2R)         =   L/(L + 2RC) 

  L 是 帧长
  R 是 单程传播时延
  C 是数据传输率
```
```
帧节 默认的单位是 字节 ， 1 字节 等于 8 比特
所以帧长 算速率时候默认乘以 8 
即使帧长 多少 B 也要乘以 【2020 统考】
```
```
停 - 等 协议： 
发送周期 =  数据帧 + 确认帧 + 数据帧 + 确认帧 
         = （数据帧 + 确认帧）X2 
```
