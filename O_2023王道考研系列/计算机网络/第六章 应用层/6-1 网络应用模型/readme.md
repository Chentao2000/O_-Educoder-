## 学习笔记:
```
客户 / 服务器：C/S
- 总是打开的叫服务器，服务来自其他被称呼客户机的 主机请求
特点
- 地位不平等，少数服务器担当整个网络的管理工作
- 客户机相互之间不通信
- 可拓展不佳
```
```
P2P 模型：
-- 对等双方直接通信，客户能够作为服务器提供资源给其他结点访问
P2P的优点在于：
- 减轻服务器压力
- 多个客户机直接共享文档
- 可拓展性好
- 网络健壮性强
缺点
- 占用很多内存
- 使得网络变得很拥挤
```
