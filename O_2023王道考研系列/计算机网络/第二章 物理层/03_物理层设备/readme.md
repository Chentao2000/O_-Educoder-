## 学习笔记:


```
10BASE5 以太网规范中 ，互相串联中继器 4个中继器 串联的5段通信介质中只有3段可以挂计算机 //  即 5-4-3 规则
```
```
放大器 放大的模拟信号 ，将衰减信号放大
中继器 放大的数字信号 ，将衰减信号整形再生
转发器 只有放大信号的功能 （为啥叫转发器？ ）
```

```
集线器 Hub 实质是一个多端口的中继器
- 集线器没有寻址的功能 ，向输出端口全部转发
从工作方式看 ，再网络中起信号放大和转发的作用 ，扩大网络的搜索范围 
Hub组合成的是共享式网络 只能在半双工的状态工作
```
