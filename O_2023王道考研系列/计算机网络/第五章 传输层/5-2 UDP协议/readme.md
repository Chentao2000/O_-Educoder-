## 刷题补充:
- 接收到有差错的数据直接丢弃
- 远程登入不适合使用UDP协议（多媒体适合）
- 通过差错检验，不能保证可靠数据传输
- UDP数据报检验不是必须的，不用设置为0 ，如果校验和结果为0 ，这全设置为1
- 伪首部值由源和目的主机仅仅在校验和计算期间建立，并不发送
